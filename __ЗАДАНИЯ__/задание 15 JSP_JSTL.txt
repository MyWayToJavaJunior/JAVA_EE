Задание:
1. Создать страницу test.jsp, на которой должны отображаться вопросы для тестирования. Каждый вопрос содержит текст вопроса и 4 варианта ответов, лишь 1 из которых верный.
На этой странице также выводится также номр текущего вопроса; кнопка "Далее", которая производит переход к следующему вопросу.
2. Создать класс Question, описывающий один вопрос. Класс Question содержит поля
List<String> answers - список текстовых вариантов ответа
int correctIndex - номер правильного варианта
3. Создать Java Bean QuizBean, реализующий логику прохождения теста.
QuizBean должен предоставлять доступ к i-му вопросу, считывать с пользователя ответ на i-й вопрос, выводить результаты тестирования в % правильных ответов.
