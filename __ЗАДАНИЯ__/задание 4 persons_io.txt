Задание на чтение/запись файлов и работу со строками:

1. Создать текстовый файл со слоедующим содержимым:

1;Фамилия1 Имя1 Отчество1;email1@email.com
2;Фамилия2 Имя2 Отчество2;email2@email.com
3;Фамилия3 Имя3 Отчество3;email1@email.com
...

2. Создать класс Person (человек) с полями
int id;
String fio, email;

3. Создать пустую коллекцию List<Person> persons.

4. Из программы на Java выполнить чтение всего содержимого файла в коллекцию строк List<string> lines

5. На основе строк в коллекции lines с помощью методов класса String заполнить коллекцию persons экземплярами класса Person.

6. Реализовать проверку данных, считанных из файла:
	а) Запись о каждом человеке обязательно должна состоять из трех частей, разделенных точкой с запятой
	б) Номер человека должен быть целым числом
	в) ФИО должно состоять из 3 частей, разделенных пробелами
	г) email должен содержать символ "@"
