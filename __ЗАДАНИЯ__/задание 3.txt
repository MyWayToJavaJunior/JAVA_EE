Задание:
1. Описать класс Waiter (официант) c полями:
String name, surname; // имя, фамилия
int age; //возраст
и переопределить метод equals таким образом, чтобы равными считались два официанта, у которых совпадает имя или разница в возрасте не превышает 5 лет.
2. Описать класс Order (заказ) с полем String content; //содержимое заказа
3. Описать класс Cafe с полем waiters (информация о всех официантах в кафе) и методом
void addOrder(Waiter w, Order o)
где Order o - ссылка на объект вновь поступившего заказа
Waiter w - ссылка на объект, описывающий требуемого клиентом официанта.
Клиент знает о требуемом официанте только имя и/или примерный возраст (+-5 лет).
На основе описанного клиентом требуемого официанта из списка всех официантов выбирается
первый, подходящий под описание, и ему назначается заказ (вывести в консоль сообщение).
Проверить работоспособность разработанных классов: заполнить список официантов, вызывать
метод addOrder с разными критериями поиска официантов.
4. В классе Cafe описать метод 
void giveTips(Waiter w)
который предназначен для выдачи чаевых описанному клиентом официанту.
Клиент знает о требуемом официанте только имя И примерный возраст (+-5 лет).
На основе описанного клиентом официанта из списка всех официантов выбирается
первый, подходящий под описание, и ему выдаются чаевые (вывести в консоль сообщение).
